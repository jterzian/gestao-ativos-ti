# 📊 Gestão de Ativos TI - Catálogo React

Aplicação front-end desenvolvida em React + Vite para gerenciamento e visualização de ativos de infraestrutura.

## 🚀 Funcionalidades
- **Listagem Dinâmica:** Exibição de ativos consumidos via simulação de API.
- **Cadastro de Ativos:** Formulário controlado para adicionar novos equipamentos.
- **Simulação de API:** Uso de `useEffect` com loader para simular carregamento real.
- **Design Minimalista:** Interface responsiva e funcional com CSS customizado.

## 🛠️ Tecnologias
- React.js
- Vite
- CSS3 (Layout Grid e Flexbox)
- Lucide React (Ícones)
